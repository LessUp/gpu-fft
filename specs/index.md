# Legacy Specs Notice

The top-level `/specs` tree is retired. The canonical specification source for this repository is now `openspec/specs/`.

## What to Use Instead

- Repository specifications: `openspec/specs/`
- Active change artifacts: `openspec/changes/`
- OpenSpec workflow: `/opsx:explore` → `/opsx:propose` → `/opsx:apply` → `/opsx:archive`

Do not add, update, or reference files under `/specs` as if they were active source-of-truth documents.
