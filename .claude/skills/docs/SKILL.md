name: docs
description: Build and preview VitePress documentation. Use when working on documentation or reviewing API docs locally.
---

## Commands

| Command | Purpose |
|---------|---------|
| `npm run docs:dev` | Start VitePress dev server with hot reload |
| `npm run docs:build` | Build static docs to `docs/.vitepress/dist/` |
| `npm run docs:preview` | Preview built docs locally |
| `npm run docs:serve` | Serve built docs via npx serve |

## Workflow

1. **Developing docs**: `npm run docs:dev` — opens dev server, edits reflect immediately
2. **Building for deploy**: `npm run docs:build` — generates static site
3. **Preview before deploy**: `npm run docs:preview` — verify build output

## Docs structure

- `docs/` — VitePress source files
- `docs/.vitepress/dist/` — built output (gitignored)
- API reference generated by TypeDoc

## Notes

- Docs auto-deploy to GitHub Pages on push to main (see `.github/workflows/pages.yml`)
- VitePress is in alpha — some features may change
