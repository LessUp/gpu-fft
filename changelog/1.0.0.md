# WebGPU FFT Library v1.0.0

**Release Date:** 2024-01-07

## 🎉 Initial Release

The first stable release of WebGPU FFT Library - a high-performance Fast Fourier Transform library with WebGPU acceleration.

### Added

#### Core FFT Engine
- **1D FFT/IFFT** - Cooley-Tukey Radix-2 DIT algorithm
  - Support for 2 to 65,536 elements
  - GPU-accelerated via WebGPU compute shaders
  - CPU fallback implementation for compatibility

- **2D FFT/IFFT** - Image processing support
  - Maximum size: 2048×2048 pixels
  - Row-column decomposition algorithm

#### GPU Resource Management
- Automatic adapter/device initialization
- Buffer pooling and caching for performance
- Pipeline caching for shader reuse
- Device loss handling and recovery

#### Frequency Domain Filtering (CPU-based)
- Low-pass filter - Remove high-frequency noise
- High-pass filter - Remove low-frequency components
- Band-pass filter - Isolate specific frequency ranges
- Filter shapes: Ideal (sharp) and Gaussian (smooth)
- Configurable cutoff frequency and bandwidth

#### Audio Spectrum Analyzer (CPU-based)
- Real-time frequency analysis for audio applications
- Hann window function for reduced spectral leakage
- dB magnitude conversion
- Configurable FFT sizes: 256, 512, 1024, 2048, 4096

#### Window Functions
- Hann window - Good for general purpose spectral analysis
- Hamming window - Reduced first sidelobe
- Blackman window - Very low sidelobes
- Flat-top window - Accurate amplitude measurement
- Rectangular window - No windowing (default)
- Apply to real or complex signals

#### Complex Number Utilities
- Arithmetic: add, subtract, multiply, scale
- Properties: magnitude, conjugate
- Conversion: interleaved ↔ complex array
- Twiddle factors: forward and inverse
- Naive DFT/IDFT for testing and validation

#### Bit-Reversal Utilities
- Bit-reversal permutation for FFT
- Power-of-2 validation
- Log2 calculation utilities

#### Error Handling
- Custom `FFTError` class with error codes
- Descriptive error messages
- Input validation with helpful feedback

#### GPU Detection
- `isWebGPUAvailable()` - Async full detection with adapter check
- `hasWebGPUSupport()` - Sync API presence check

#### TypeScript Support
- Full type definitions for all APIs
- Strict mode compatible
- ESM and CJS exports

#### Testing & Quality
- 106 tests covering all functionality
- Property-based tests with fast-check
- Unit tests with Vitest
- V8 code coverage reporting

#### Documentation
- Comprehensive README with quick start
- API reference via TypeDoc
- Code examples collection
- Interactive demos

### Technical Details

#### Implementation
- **Algorithm:** Cooley-Tukey Radix-2 Decimation-In-Time (DIT)
- **Shader Language:** WGSL (WebGPU Shading Language)
- **Workgroup Size:** 256 (fixed for current implementation)
- **Memory Layout:** Interleaved complex numbers [real, imag, real, imag, ...]

#### Performance Optimizations
- Batched command submission
- Buffer reuse for same-size transforms
- Compiled shader caching
- Parallel execution via GPU compute shaders

### Known Limitations

- GPU shader only supports `workgroupSize: 256`
- Spectrum analyzer uses CPU FFT internally
- Maximum 1D size: 65,536 elements
- Maximum 2D size: 2048×2048
