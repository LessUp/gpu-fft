# Changelog Directory

This directory contains detailed changelogs for each version of the WebGPU FFT Library.

## Structure

- Each version has its own file (e.g., `1.0.0.md`, `1.1.0.md`)
- The root `CHANGELOG.md` contains a summary of all changes
- This follows the [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format

## Available Versions

| Version | Date | Description |
|---------|------|-------------|
| [1.1.0](1.1.0.md) | 2026-04-16 | Documentation overhaul with bilingual support |
| [1.0.1](1.0.1.md) | 2026-04-16 | Security updates and dependency upgrades |
| [1.0.0](1.0.0.md) | 2024-01-07 | Initial release with GPU acceleration |

## Categories

Each changelog entry is categorized as one of:

- **Added** - New features
- **Changed** - Changes to existing functionality
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Security-related changes
