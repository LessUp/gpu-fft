浏览器兼容性

WebGPU FFT Library 的 GPU 加速功能需要 WebGPU 支持。以下是主要浏览器的兼容性矩阵。

浏览器最低版本WebGPU 状态备注
Chrome113+✅ 稳定版Windows、macOS、Linux 完整支持
Edge113+✅ 稳定版与 Chrome 相同（基于 Chromium）
Firefox128+✅ 稳定版自版本 128 起默认启用
Safari17+⚠️ 预览版仅技术预览版
Opera99+✅ 稳定版基于 Chromium，与 Chrome 支持相同
移动版 Chrome121+✅ 稳定版仅 Android
移动版 Safari-❌ 不支持暂不支持 WebGPU

特性检测

在使用 GPU 功能前务必检查 WebGPU 支持：

import { isWebGPUAvailable } from 'webgpu-fft';

if (await isWebGPUAvailable()) {
  // 使用 GPU 加速 FFT
  const engine = await createFFTEngine();
} else {
  // 回退到 CPU 实现
  const spectrum = cpuFFT(input);
}
