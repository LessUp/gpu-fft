故障排除

您可能遇到的常见问题及解决方案。

WebGPU 不可用

FFTError: WebGPU is not available

当浏览器不支持 WebGPU 或 WebGPU 被禁用时会出现此错误。

解决方案：
  • 在 浏览器兼容性 中检查浏览器支持情况
  • 将浏览器更新到最新版本（Chrome 113+、Edge 113+、Firefox 128+）
  • 使用 CPU 回退：cpuFFT() / cpuIFFT()

输入大小无效

FFTError: Input size must be a power of 2

FFT 要求输入长度必须是 2 的幂（2、4、8、16、...、65536）。

解决方案：
  • 将数据填充到下一个 2 的幂
  • 调用 FFT 前使用 validateFFTInput() 验证
  • 检查输入数组长度（对于 N 个复数必须是 2×N）

输入过大

FFTError: Input size exceeds maximum of 65536

支持的最大输入大小为 65,536 个复数。

解决方案：
  • 将大数据集拆分为较小的块
  • 对连续信号使用重叠窗口技术
  • 对于非常大的数组，考虑使用 CPU 实现

性能提示

  • 复用引擎： 创建一次 FFTEngine 并多次复用进行变换
  • 释放资源： 完成后务必调用 dispose() 释放 GPU 内存
  • 小型数据使用 CPU： 对于小于 1024 元素的变换，CPU 可能更快
  • 批处理： 一起处理多个相同大小的变换
