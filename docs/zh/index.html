简介

WebGPU FFT Library 是一个高性能的快速傅里叶变换（FFT）库，使用 WebGPU 计算着色器进行 GPU 加速，同时提供完整的 CPU 回退实现。它支持一维和二维变换， 非常适合信号处理、音频分析和图像滤波等应用场景。

本库在设计上兼顾性能和易用性。无论是构建实时音频频谱分析仪，还是实现频域图像滤波器， WebGPU FFT Library 都能为您提供所需的工具。

特性

  • ⚡
    GPU 加速 WebGPU 计算着色器实现最高性能
  • 🔄
    CPU 回退 无需 WebGPU 支持也可完整运行
  • 📊
    1D FFT/IFFT 支持 2 到 65,536 个元素
  • 🖼️
    2D FFT/IFFT 图像处理支持最大 2048×2048 像素
  • 🎵
    音频频谱分析 实时频率分析，支持窗函数
  • 🔍
    频域滤波 低通、高通、带通滤波器
  • 🔷
    TypeScript 支持 完整类型定义，支持严格模式
  • 🚀
    零依赖 无运行时依赖，包体积最小

快速开始

安装

npm install webgpu-fft

基本用法（GPU 路径）

import { createFFTEngine, isWebGPUAvailable } from 'webgpu-fft';

// 检查 WebGPU 支持
if (await isWebGPUAvailable()) {
  const engine = await createFFTEngine();
  
  // 准备输入（交错格式：[实部, 虚部, ...]）
  const input = new Float32Array(16);
  for (let i = 0; i < 8; i++) {
    input[i * 2] = Math.sin(i);     // 实部
    input[i * 2 + 1] = 0;               // 虚部
  }
  
  const spectrum = await engine.fft(input);
  const recovered = await engine.ifft(spectrum);
  
  engine.dispose(); // 释放 GPU 资源
}

CPU 回退

import { cpuFFT, cpuIFFT } from 'webgpu-fft';

const input = new Float32Array([1, 0, 2, 0, 3, 0, 4, 0]);
const spectrum = cpuFFT(input);
const recovered = cpuIFFT(spectrum);
