FFT 引擎

createFFTEngine
async function createFFTEngine(
  config?: Partial<FFTEngineConfig>
): Promise<FFTEngine>

创建具有 GPU 加速的新 FFT 引擎实例。返回一个提供 FFT/IFFT 方法并管理 GPU 资源的 FFTEngine 对象。

参数类型必需描述
configPartial<FFTEngineConfig> 可选引擎配置选项
config.workgroupSizenumber 可选计算着色器 workgroup 大小（默认：256，目前仅支持 256）
返回
Promise<FFTEngine>

一个 Promise，解析为 FFTEngine 实例

💡 示例
const engine = await createFFTEngine();
const spectrum = await engine.fft(inputData);
engine.dispose();
interface FFTEngine
interface FFTEngine {
  // 一维变换
  fft(input: Float32Array): Promise<Float32Array>;
  ifft(input: Float32Array): Promise<Float32Array>;

  // 二维变换
  fft2d(input: Float32Array, width: number, height: number): Promise<Float32Array>;
  ifft2d(input: Float32Array, width: number, height: number): Promise<Float32Array>;

  // 资源管理
  dispose(): void;
}

CPU FFT

cpuFFT / cpuIFFT
function cpuFFT(input: Float32Array): Float32Array

同步 CPU 实现的前向和逆 FFT。无需 WebGPU 即可工作。输入必须是交错复数 [实部0, 虚部0, 实部1, 虚部1, ...]，长度为 2 的幂。

参数类型必需描述
inputFloat32Array 必需交错复数输入（长度必须是 2 的幂 × 2）
返回
Float32Array

交错复数频域数据

💡 示例
const input = new Float32Array([1, 0, 2, 0, 3, 0, 4, 0]);
const spectrum = cpuFFT(input);
const recovered = cpuIFFT(spectrum);
cpuFFT2D / cpuIFFT2D
function cpuFFT2D(
  input: Float32Array,
  width: number,
  height: number
): Float32Array

用于图像处理的二维 FFT，使用行列分解。最大尺寸：2048×2048。

💡 示例
const spectrum2d = cpuFFT2D(imageData, 512, 512);
const imageBack = cpuIFFT2D(spectrum2d, 512, 512);

图像滤波器

createImageFilter
async function createImageFilter(
  config: ImageFilterConfig
): Promise<ImageFilter>

创建频域图像滤波器。支持低通、高通和带通滤波器，可选择理想（Ideal）或高斯（Gaussian）形状。

属性类型描述
type'lowpass' | 'highpass' | 'bandpass'滤波器类型
shape'ideal' | 'gaussian'滤波器形状（锐边缘或平滑衰减）
cutoffFrequencynumber截止频率（0.0 到 1.0，归一化）
bandwidthnumber仅用于带通：滤波器带宽

GPU 检测

isWebGPUAvailable / hasWebGPUSupport
async function isWebGPUAvailable(): Promise<boolean>
function hasWebGPUSupport(): boolean
⚠️ 注意

isWebGPUAvailable() 异步检查适配器是否可用，而 hasWebGPUSupport() 仅检查 API 是否存在。在创建引擎之前务必使用 isWebGPUAvailable()。

💡 示例
if (await isWebGPUAvailable()) {
  // 使用 GPU 引擎
  const engine = await createFFTEngine();
} else {
  // 回退到 CPU
  const spectrum = cpuFFT(input);
}
