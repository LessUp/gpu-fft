快速入门指南

在几分钟内开始使用 WebGPU FFT Library。本指南将带您完成安装、基础概念和第一个 FFT 变换。

1

安装

通过 npm 安装库：

npm install webgpu-fft
ℹ️

要求： Node.js 18+ 和支持 WebGPU 的浏览器（Chrome 113+、 Edge 113+、Firefox 128+、Safari 17+）。

2

检查 WebGPU 可用性

在使用 GPU 加速的 FFT 之前，先检查 WebGPU 是否可用。库提供了一个异步函数来完成这个检查： 

import { isWebGPUAvailable } from 'webgpu-fft';

if (await isWebGPUAvailable()) {
  console.log('WebGPU 可用！');
} else {
  console.log('使用 CPU 回退');
}
3

创建 FFT 引擎

创建 FFT 引擎实例。引擎负责管理 GPU 资源并提供 FFT/IFFT 方法： 

import { createFFTEngine } from 'webgpu-fft';

// 使用默认配置创建引擎
const engine = await createFFTEngine();

// 或自定义 workgroup 大小
const engine = await createFFTEngine({
  workgroupSize: 256  // 当前限制：仅支持 256
});
4

准备输入数据

库使用交错复数格式：[实部0, 虚部0, 实部1, 虚部1, ...]。 输入大小必须是 2 的幂（2、4、8、...、65536）。 

// 创建一个简单的正弦波信号
const size = 1024;
const frequency = 5;
const input = new Float32Array(size * 2);

for (let i = 0; i < size; i++) {
  input[i * 2] = Math.sin((2 * 3.14159 * frequency * i) / size);  // 实部
  input[i * 2 + 1] = 0;  // 虚部
}
💡

提示： 对于小型变换（< 1024 元素），使用 cpuFFT 可能更快，因为开销更小。

5

计算 FFT

执行前向 FFT 变换：

// 计算 FFT
const spectrum = await engine.fft(input);

// spectrum 现在包含复数频率数据
// spectrum[i*2] = 第 i 个频率 bin 的实部
// spectrum[i*2+1] = 第 i 个频率 bin 的虚部

// 获取幅度
const magnitude = Math.sqrt(
  spectrum[i * 2] * spectrum[i * 2] + 
  spectrum[i * 2 + 1] * spectrum[i * 2 + 1]
);
6

计算 IFFT（逆 FFT）

变换回时域：

// 计算逆 FFT
const recovered = await engine.ifft(spectrum);

// 恢复的信号应该与原始输入匹配
// （在浮点精度范围内）
7

清理资源

完成后始终释放引擎以释放 GPU 资源。这在单页应用中尤为重要： 

// 释放所有 GPU 资源
engine.dispose();

CPU 回退

如果 WebGPU 不可用，请使用 CPU 实现：

import { cpuFFT, cpuIFFT } from 'webgpu-fft';

const input = new Float32Array([1, 0, 2, 0, 3, 0, 4, 0]);
const spectrum = cpuFFT(input);
const recovered = cpuIFFT(spectrum);
💡

性能提示： 对于图像的 2D FFT，CPU 实现使用行列分解算法， 可以处理最大 2048×2048 像素的图像。

完整示例

以下是一个完整的可运行示例：

import { 
  createFFTEngine, 
  isWebGPUAvailable,
  cpuFFT,
  cpuIFFT 
} from 'webgpu-fft';

async function fftExample() {
  // 创建测试信号：两个正弦波的叠加
  const size = 1024;
  const input = new Float32Array(size * 2);
  
  for (let i = 0; i < size; i++) {
    const t = i / size;
    input[i * 2] = Math.sin(2 * Math.PI * 10 * t) + 
                   Math.sin(2 * Math.PI * 20 * t);
    input[i * 2 + 1] = 0;
  }
  
  // 优先尝试 GPU，回退到 CPU
  if (await isWebGPUAvailable()) {
    console.log('使用 GPU 加速');
    const engine = await createFFTEngine();
    
    const spectrum = await engine.fft(input);
    const recovered = await engine.ifft(spectrum);
    
    // 验证往返精度
    let maxError = 0;
    for (let i = 0; i < input.length; i++) {
      maxError = Math.max(maxError, Math.abs(input[i] - recovered[i]));
    }
    console.log(`最大往返误差: ${maxError}`);  // ~1e-6
    
    engine.dispose();
  } else {
    console.log('使用 CPU 回退');
    const spectrum = cpuFFT(input);
    const recovered = cpuIFFT(spectrum);
  }
}

fftExample();
