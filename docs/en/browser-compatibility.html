Browser Compatibility

WebGPU FFT Library requires WebGPU support for GPU acceleration. Below is the compatibility matrix for major browsers.

Feature Detection

Always check for WebGPU support before using GPU features:

import { isWebGPUAvailable } from 'webgpu-fft';

if (await isWebGPUAvailable()) {
  // Use GPU-accelerated FFT
  const engine = await createFFTEngine();
} else {
  // Fall back to CPU implementation
  const spectrum = cpuFFT(input);
}
