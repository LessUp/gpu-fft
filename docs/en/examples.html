🎵 Audio Spectrum Analysis

Audio CPU

Analyze audio signals to detect frequency components in real-time.

import { createSpectrumAnalyzer } from 'webgpu-fft';

const analyzer = await createSpectrumAnalyzer({
  fftSize: 2048,
  sampleRate: 44100
});

// Analyze audio data
const audioData = new Float32Array(2048);
// ... fill with audio samples ...

const spectrumDb = await analyzer.analyze(audioData);
// spectrumDb[i] = dB magnitude for frequency bin i

// Get frequencies for bins
const freqs = analyzer.getFrequencies();
console.log(`Peak at ${freqs[10]} Hz: ${spectrumDb[10].toFixed(1)} dB`);

analyzer.dispose();

🖼️ Image Low-Pass Filter

CPU

Apply a Gaussian low-pass filter to an image using frequency domain processing.

import { createImageFilter, cpuFFT2D, cpuIFFT2D } from 'webgpu-fft';

// Create a Gaussian low-pass filter
const filter = await createImageFilter({
  type: 'lowpass',
  shape: 'gaussian',
  cutoffFrequency: 0.2  // Keep low frequencies, blur image
});

// Convert image to complex format [r,i,r,i,...]
const width = 512, height = 512;
const imageData = new Float32Array(width * height * 2);
// ... fill imageData with grayscale pixel values ...

// Apply filter
const filtered = await filter.apply(imageData, width, height);
// filtered contains blurred image

filter.dispose();

⚡ GPU Batch Processing

GPU

Process multiple FFT transforms efficiently using GPU batching.

import { createFFTEngine } from 'webgpu-fft';

const engine = await createFFTEngine();

// Process multiple signals with same size
const signals = [];
for (let i = 0; i < 100; i++) {
  signals.push(generateSignal(i));
}

// Batch process - reuses GPU buffers
const results = await Promise.all(
  signals.map(async (sig) => engine.fft(sig))
);

engine.dispose();
