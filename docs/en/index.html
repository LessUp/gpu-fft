Introduction

The WebGPU FFT Library provides high-performance Fast Fourier Transform (FFT) calculations using WebGPU compute shaders for GPU acceleration, with a full CPU fallback implementation. It supports both 1D and 2D transforms, making it ideal for signal processing, audio analysis, and image filtering applications.

This library is designed with performance and ease of use in mind. Whether you're building a real-time audio spectrum analyzer or implementing frequency-domain image filters, WebGPU FFT Library provides the tools you need.

Features

  • ⚡
    GPU Acceleration WebGPU compute shaders for maximum performance
  • 🔄
    CPU Fallback Full implementation works without WebGPU support
  • 📊
    1D FFT/IFFT Support for 2 to 65,536 elements
  • 🖼️
    2D FFT/IFFT Image processing up to 2048×2048 pixels
  • 🎵
    Audio Spectrum Analysis Real-time frequency analysis with window functions
  • 🔍
    Frequency Filtering Low-pass, high-pass, band-pass filters
  • 🔷
    TypeScript Support Full type definitions with strict mode
  • 🚀
    Zero Dependencies No runtime dependencies for minimal bundle size

Quick Start

Installation

npm install webgpu-fft

Basic Usage (GPU Path)

import { createFFTEngine, isWebGPUAvailable } from 'webgpu-fft';

// Check WebGPU support
if (await isWebGPUAvailable()) {
  const engine = await createFFTEngine();
  
  // Prepare input (interleaved: [real, imag, ...])
  const input = new Float32Array(16);
  for (let i = 0; i < 8; i++) {
    input[i * 2] = Math.sin(i);     // Real part
    input[i * 2 + 1] = 0;               // Imaginary part
  }
  
  const spectrum = await engine.fft(input);
  const recovered = await engine.ifft(spectrum);
  
  engine.dispose(); // Clean up GPU resources
}

CPU Fallback

import { cpuFFT, cpuIFFT } from 'webgpu-fft';

const input = new Float32Array([1, 0, 2, 0, 3, 0, 4, 0]);
const spectrum = cpuFFT(input);
const recovered = cpuIFFT(spectrum);
