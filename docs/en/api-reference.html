FFT Engine

createFFTEngine
async function createFFTEngine(
  config?: Partial<FFTEngineConfig>
): Promise<FFTEngine>

Creates a new FFT engine instance with GPU acceleration. Returns an FFTEngine object that provides FFT/IFFT methods and manages GPU resources.

Parameter Type Required Description
config Partial<FFTEngineConfig> Optional Engine configuration options
config.workgroupSize number Optional Compute shader workgroup size (default: 256, currently only 256 supported)
Returns
Promise<FFTEngine>

A Promise that resolves to an FFTEngine instance

💡 Example
const engine = await createFFTEngine();
const spectrum = await engine.fft(inputData);
engine.dispose();
interface FFTEngine
interface FFTEngine {
  // 1D transforms
  fft(input: Float32Array): Promise<Float32Array>;
  ifft(input: Float32Array): Promise<Float32Array>;

  // 2D transforms
  fft2d(input: Float32Array, width: number, height: number): Promise<Float32Array>;
  ifft2d(input: Float32Array, width: number, height: number): Promise<Float32Array>;

  // Resource management
  dispose(): void;
}

CPU FFT

cpuFFT / cpuIFFT
function cpuFFT(input: Float32Array): Float32Array

Synchronous CPU implementation of forward and inverse FFT. Works without WebGPU. Input must be interleaved complex numbers [real0, imag0, real1, imag1, ...] with power-of-2 length.

Parameter Type Required Description
input Float32Array Required Interleaved complex input (length must be power of 2 × 2)
Returns
Float32Array

Interleaved complex frequency domain data

💡 Example
const input = new Float32Array([1, 0, 2, 0, 3, 0, 4, 0]);
const spectrum = cpuFFT(input);
const recovered = cpuIFFT(spectrum);
cpuFFT2D / cpuIFFT2D
function cpuFFT2D(
  input: Float32Array,
  width: number,
  height: number
): Float32Array

2D FFT for image processing using row-column decomposition. Maximum size: 2048×2048.

💡 Example
const spectrum2d = cpuFFT2D(imageData, 512, 512);
const imageBack = cpuIFFT2D(spectrum2d, 512, 512);

GPU Detection

isWebGPUAvailable / hasWebGPUSupport
async function isWebGPUAvailable(): Promise<boolean>
function hasWebGPUSupport(): boolean
⚠️ Note

isWebGPUAvailable() asynchronously checks if an adapter is available, while hasWebGPUSupport() only checks if the API is present. Always use isWebGPUAvailable() before creating an engine.

💡 Example
if (await isWebGPUAvailable()) {
  // Use GPU engine
  const engine = await createFFTEngine();
} else {
  // Fallback to CPU
  const spectrum = cpuFFT(input);
}
