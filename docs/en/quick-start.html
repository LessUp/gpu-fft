Quick Start Guide

Get started with WebGPU FFT Library in minutes. This guide will walk you through installation, basic concepts, and your first FFT transform.

1

Installation

Install the library via npm:

npm install webgpu-fft
ℹ️

Requirements: Node.js 18+ and a WebGPU-capable browser (Chrome 113+, Edge 113+, Firefox 128+, Safari 17+).

2

Check WebGPU Availability

Before using the GPU-accelerated FFT, check if WebGPU is available. The library provides an async function for this: 

import { isWebGPUAvailable } from 'webgpu-fft';

if (await isWebGPUAvailable()) {
  console.log('WebGPU is available!');
} else {
  console.log('Using CPU fallback');
}
3

Create FFT Engine

Create an FFT engine instance. The engine handles GPU resources and provides FFT/IFFT methods: 

import { createFFTEngine } from 'webgpu-fft';

// Create engine with default configuration
const engine = await createFFTEngine();

// Or with custom workgroup size
const engine = await createFFTEngine({
  workgroupSize: 256  // Current limitation: only 256 supported
});
4

Prepare Input Data

The library uses interleaved complex number format: [real0, imag0, real1, imag1, ...]. Input size must be power of 2 (2, 4, 8, ..., 65536). 

// Create a simple sine wave signal
const size = 1024;
const frequency = 5;
const input = new Float32Array(size * 2);

for (let i = 0; i < size; i++) {
  input[i * 2] = Math.sin((2 * 3.14159 * frequency * i) / size);  // Real
  input[i * 2 + 1] = 0;  // Imaginary
}
💡

Tip: Use cpuFFT for small transforms (< 1024 elements) as it may be faster due to lower overhead.

5

Compute FFT

Perform the forward FFT transform:

// Compute FFT
const spectrum = await engine.fft(input);

// spectrum now contains complex frequency data
// spectrum[i*2] = real part of i-th frequency bin
// spectrum[i*2+1] = imaginary part of i-th frequency bin

// To get magnitude
const magnitude = Math.sqrt(
  spectrum[i * 2] * spectrum[i * 2] + 
  spectrum[i * 2 + 1] * spectrum[i * 2 + 1]
);
6

Compute IFFT (Inverse FFT)

To transform back to time domain:

// Compute inverse FFT
const recovered = await engine.ifft(spectrum);

// The recovered signal should match the original input
// (within floating-point precision)
7

Cleanup

Always dispose the engine when done to release GPU resources. This is especially important in single-page applications: 

// Release all GPU resources
engine.dispose();

CPU Fallback

If WebGPU is not available, use the CPU implementation:

import { cpuFFT, cpuIFFT } from 'webgpu-fft';

const input = new Float32Array([1, 0, 2, 0, 3, 0, 4, 0]);
const spectrum = cpuFFT(input);
const recovered = cpuIFFT(spectrum);
💡

Performance Tip: For 2D FFT on images, the CPU implementation uses the row-column decomposition algorithm and can handle images up to 2048×2048 pixels.

Complete Example

Here's a complete working example:

import { 
  createFFTEngine, 
  isWebGPUAvailable,
  cpuFFT,
  cpuIFFT 
} from 'webgpu-fft';

async function fftExample() {
  // Create test signal: sum of two sine waves
  const size = 1024;
  const input = new Float32Array(size * 2);
  
  for (let i = 0; i < size; i++) {
    const t = i / size;
    input[i * 2] = Math.sin(2 * Math.PI * 10 * t) + 
                   Math.sin(2 * Math.PI * 20 * t);
    input[i * 2 + 1] = 0;
  }
  
  // Try GPU first, fallback to CPU
  if (await isWebGPUAvailable()) {
    console.log('Using GPU acceleration');
    const engine = await createFFTEngine();
    
    const spectrum = await engine.fft(input);
    const recovered = await engine.ifft(spectrum);
    
    // Verify round-trip accuracy
    let maxError = 0;
    for (let i = 0; i < input.length; i++) {
      maxError = Math.max(maxError, Math.abs(input[i] - recovered[i]));
    }
    console.log(`Max round-trip error: ${maxError}`);  // ~1e-6
    
    engine.dispose();
  } else {
    console.log('Using CPU fallback');
    const spectrum = cpuFFT(input);
    const recovered = cpuIFFT(spectrum);
  }
}

fftExample();
