# Learning Resources

Curated resources for learning about FFT algorithms, WebGPU, and digital signal processing.

## Interactive Visualizations

<div class="reference-grid">
<div class="reference-card">
<div class="reference-card-title">[3Blue1Brown: Fourier Transform](https://www.3blue1brown.com/lessons/fourier-series)</div>
<div class="reference-card-desc">
Grant Sanderson's excellent visual introduction to Fourier series and transforms. Intuitive explanations with beautiful animations.
</div>
<div class="reference-card-link">Video · Free</div>
</div>

<div class="reference-card">
<div class="reference-card-title">[Seeing Theory: Fourier Transform](https://seeing-theory.brown.edu/fourier-transform/index.html)</div>
<div class="reference-card-desc">
Interactive visualization from Brown University. Experiment with different signals and see their Fourier transforms in real-time.
</div>
<div class="reference-card-link">Interactive · Free</div>
</div>

<div class="reference-card">
<div class="reference-card-title">[Better Explained: Interactive FFT](https://betterexplained.com/articles/an-interactive-guide-to-the-fourier-transform/)</div>
<div class="reference-card-desc">
Kalid Azad's intuitive guide with interactive examples. Focuses on building understanding rather than formal derivation.
</div>
<div class="reference-card-link">Article · Free</div>
</div>

<div class="reference-card">
<div class="reference-card-title">[FFT Playground](/playground/index)</div>
<div class="reference-card-desc">
Our built-in interactive FFT explorer. Experiment with different input signals and see real-time FFT results.
</div>
<div class="reference-card-link">Interactive · Built-in</div>
</div>
</div>

<div class="reference-grid">

## WebGPU & GPU Computing

<div class="reference-grid">
<div class="reference-card">
<div class="reference-card-title">[WebGPU Fundamentals](https://webgpufundamentals.org/)</div>
<div class="reference-card-desc">
Comprehensive WebGPU tutorials by Gregg Tavares. Covers compute shaders, buffers, and GPU programming patterns.
</div>
<div class="reference-card-link">Tutorial · Free</div>
</div>

<div class="reference-card">
<div class="reference-card-title">[WebGPU by Example](https://webgpu.github.io/webgpu-by-example/)</div>
<div class="reference-card-desc">
Official WebGPU working group examples. Reference implementations for common GPU patterns.
</div>
<div class="reference-card-link">Examples · Free</div>
</div>

<div class="reference-card">
<div class="reference-card-title">[Compute Shaders: A Gentle Introduction](https://www.3dgep.com/introduction-to-compute-shaders/)</div>
<div class="reference-card-desc">
Detailed introduction to GPU compute programming. While focused on CUDA/DirectX, concepts transfer to WebGPU.
</div>
<div class="reference-card-link">Tutorial · Free</div>
</div>

<div class="reference-card">
<div class="reference-card-title">[GPU Gems 2: Chapter 37](https://developer.nvidia.com/gpugems/gpugems2/part-iv-image-processing/chapter-37-fast-fourier-transform-gpu)</div>
<div class="reference-card-desc">
NVIDIA's GPU Gems chapter on FFT implementation. Classic reference for GPU FFT optimization techniques.
</div>
<div class="reference-card-link">Book Chapter · Free</div>
</div>
</div>

## Textbooks

<div class="reference-grid">
<div class="reference-card">

<div class="reference-card-desc">
Richard Lyons' approachable DSP textbook. Excellent for self-study with minimal prerequisites.
</div>
<div class="reference-card-link">Book · Paid</div>
</div>
