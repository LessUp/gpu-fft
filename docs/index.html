npm 包

通过 npm install webgpu-fft 安装，支持 ESM 和 CommonJS 两种导入方式。

Package

核心特性

  • ✅ 1D FFT/IFFT — 支持 2 到 65,536 个元素
  • ✅ 2D FFT/IFFT — 最大 2048×2048
  • ✅ Bank Conflict 优化 — 消除 Shared Memory 访问冲突
  • ✅ 频域滤波 — 低通/高通/带通（Ideal + Gaussian）
  • ✅ 实时频谱分析 — 支持麦克风/音频文件输入
  • ✅ 窗函数 — Hann、Hamming、Blackman、Flat-top
  • ✅ CPU 回退 — 无需 GPU 也可使用完整 FFT
  • ✅ 属性测试 — fast-check 全面正确性验证

快速开始

// 安装
npm install webgpu-fft

// GPU 路径
import { createFFTEngine, isWebGPUAvailable } from 'webgpu-fft';

if (await isWebGPUAvailable()) {
  const engine = await createFFTEngine();
  const result = await engine.fft(inputData);
  engine.dispose();
}

// CPU 回退路径
import { cpuFFT, cpuIFFT } from 'webgpu-fft';
const spectrum = cpuFFT(signal);
