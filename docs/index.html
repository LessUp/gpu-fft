WebGPU FFT Library

High-performance GPU-accelerated Fast Fourier Transform library with WebGPU compute shaders. Supports 1D/2D FFT, frequency domain filtering, and real-time spectrum analysis.

📦 v1.1.0 🔷 TypeScript 5.9 ⚡ WebGPU 📜 MIT License 🚀 Zero Dependencies
