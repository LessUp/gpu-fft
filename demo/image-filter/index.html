How It Works

This demo applies frequency-domain filtering to images using an in-page FFT implementation. It is an interactive showcase and does not fully mirror the packaged library runtime:

  1. FFT: Transform the image from spatial domain to frequency domain
  2. Filter: Multiply by a filter mask to attenuate certain frequencies
  3. IFFT: Transform back to spatial domain to get the filtered image

Filter Types

  • Low-Pass: Removes high frequencies → Blurs the image
  • High-Pass: Removes low frequencies → Enhances edges

Filter Shapes

  • Gaussian: Smooth rolloff, no ringing artifacts
  • Ideal: Sharp cutoff, may cause ringing (Gibbs phenomenon)
