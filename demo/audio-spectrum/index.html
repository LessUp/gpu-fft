How It Works

This demo captures audio from your microphone or an audio file, then visualizes its frequency content in real time. The current demo page uses the Web Audio API for visualization rather than the packaged FFT runtime, so treat it as an interactive showcase instead of a direct runtime mirror.

Controls

  • Microphone: Uses your device's microphone for live audio input
  • Audio File: Analyze any audio file from your computer
  • Test Tone: Generate a pure sine wave at a specific frequency
  • FFT Size: Larger sizes give better frequency resolution but slower updates
